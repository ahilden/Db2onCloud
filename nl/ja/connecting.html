接続

コマンド行インターフェース、IBM® またはサード・パーティーのアプリケーションやツール、作成したアプリケーションを DB2® データベースに接続できます。

前提条件

接続で問題が生じる場合は、前提条件 を満たすようにしてください。

環境の構成

ローカル・アプリケーションやツールを DB2 データベースに接続するには、環境の構成 を行う必要があります。

プログラムによる接続

共通プログラミング言語を使用して、DB2 データベースに接続するアプリケーションを作成できます。

  • JDBC
  • ODBC
  • .NET
  • PHP

アプリケーションとツールの接続

また、外部アプリケーションやツールを Db2 on Cloud に接続して利用し、 データをさらに管理または分析することも可能です。 例えば、以下を行えます。

  • 分析データベースを必要とする IBM® Cloud アプリケーションを接続します。
  • IBM Watson™ Studio (以前の IBM Data Science Experience) から接続します。
  • IBM InfoSphere® Data Architect を接続して、データベース・スキーマを設計およびデプロイします。
  • IBM Cognos® サーバーを接続して、データに対して Cognos レポートを実行します。
  • SQL ベースのツール (Tableau、Microsoft Excel など) を接続して、データを操作、分析、または視覚化します。
    • Tableau
    • Microsoft Excel
  • Aginity Workbench を接続して、Netezza® データ・モデルおよびデータを Db2 Warehouse on Cloud にマイグレーションします。
