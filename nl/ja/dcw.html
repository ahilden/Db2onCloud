データベース互換性の評価

IBM Database Conversion Workbench (DCW) は、IBM Data Studio にデータベース・マイグレーション機能を追加する無料のプラグインです。 IBM Data Studio と Database Conversion Workbench は、いずれも Web コンソールからダ ウンロードできます。

ご使用のデータベースと DB2 との互換性を評価し、SQL を変換するに あたって、 IBM Database Harmony Profiler という軽量の無料ツールを使用することもできます。

Database Conversion Workbench と Database Harmony Profiler について詳しくは、IBMDatabase Conversion Workbench (DCW) を参照してください。
