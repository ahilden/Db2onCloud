Migrazione dei dati su IBM Cloud

Puoi caricare i dati da un file di dati in un formato delimitato (CSV o TXT) situato su una rete locale o in un archivio oggetti (Amazon S3 o IBM Cloud Object Storage) su IBM® Db2® on Cloud. Puoi anche migrare i tuoi dati da un sistema installato in loco.

Caricamento dei dati dall'archivio oggetti

Per caricare i dati da Amazon S3, seleziona uno dei seguenti metodi:

  • Console web Db2 on Cloud. Load > Amazon S3.

  • Direttamente dalle tabelle esterne. Il seguente è un esempio di istruzione SQL:

    INSERT INTO <table-name> SELECT * FROM EXTERNAL '<mys3file.txt>' USING
    (CCSID 1208 s3('s3.amazonaws.com', 
    '<S3-access-key-ID>',
    '<S3-secret-access-key>', 
    '<my_bucket>'
       )
  )

Per caricare i dati da IBM Cloud Object Storage utilizzando direttamente tabelle esterne, puoi utilizzare la seguente istruzione SQL di esempio:

INSERT INTO <table-name> SELECT * FROM EXTERNAL '<mys3file.txt>' USING
  (CCSID 1208 s3('s3-api.us-geo.objectstorage.softlayer.net', 
  '<S3-access-key-ID>',
  '<S3-secret-access-key>', 
  '<my_bucket>'
     )
  )

Per IBM Cloud Object Storage, per creare le credenziali HMAC durante la creazione di nuove credenziali di servizio, specifica {"HMAC:true"} nel campo Aggiungi parametri di configurazione inline.

Migrazione dei dati dal sistema installato in loco

Per migrare i tuoi dati da un sistema installato in loco, scegli uno dei seguenti metodi in base alla dimensione del tuo dataset:

  • Meno di 25 TB di dati: CLI IBM Lift
  • 25 o più TB di dati: IBM Cloud Mass Data Migration

CLI Lift

La CLI Lift è un'applicazione che puoi utilizzare gratuitamente per migrare i tuoi dati su IBM Cloud dalle diverse origini dati elencate nella Tabella 1.

IBM Cloud Mass Data Migration

Un modo veloce, semplice e sicuro per trasferire fisicamente da terabyte a petabyte di dati in IBM Cloud. Mass Data Migration è un dispositivo di archiviazione mobile con 120 TB di capacità di archiviazione utilizzabile per accelerare lo spostamento dei dati in IBM Cloud. Supera le comuni sfide di trasferimento come ad esempio costi elevati, lunghi tempi di trasferimento e problemi di sicurezza, tutto in un unico servizio.

Per ulteriori informazioni sul dispositivo Mass Data Migration, vedi: Introduzione a IBM Cloud Mass Data Migration.
