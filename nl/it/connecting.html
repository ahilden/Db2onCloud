Collegamento

Puoi collegare interfacce della riga di comando, strumenti o applicazioni di terze parti o di IBM® o applicazioni che hai creato nel tuo database Db2®.

Prerequisiti

Se stai avendo dei problemi di connessione, completa i prerequisiti .

Configurazione dell'ambiente

Per collegare gli strumenti e le applicazioni locali al tuo database Db2, devi configurare il tuo ambiente .

Collegamento in modo programmatico

Puoi utilizzare linguaggi di programmazione comuni per creare applicazioni che si collegano a un database Db2.

  • JDBC
  • ODBC
  • .NET
  • PHP

Collegamento di applicazioni e strumenti

Puoi anche connettere applicazioni e strumenti esterni a Db2 on Cloud e utilizzarli per un'ulteriore gestione o analisi dei dati. Ad esempio:

  • Connetti le tue applicazioni IBM Cloud™ che necessitano di un database di analisi.
  • Connessione da IBM Watson™ Studio (in precedenza IBM Data Science Experience).
  • Connetti IBM InfoSphere® Data Architect per progettare e distribuire il tuo schema di database.
  • Connetti un server IBM Cognos® per eseguire i report Cognos sui tuoi dati.
  • Connetti gli strumenti basati su SQL quali Tableau o Microsoft Excel per manipolare, analizzare o visualizzare i tuoi dati.
    • Tableau
    • Microsoft Excel
  • Connect Aginity Workbench to migrate Netezza® data models and data to Db2 Warehouse on Cloud.
