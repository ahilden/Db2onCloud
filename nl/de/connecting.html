Verbindung herstellen

Sie können eine Verbindung von Befehlszeilenschnittstellen, von IBM® Anwendungen oder Tools, von Anwendungen oder Tools eines anderen Anbieters oder von Apps, die Sie erstellen, zu Ihrer Db2®-Datenbank herstellen.

Voraussetzungen

Falls beim Herstellen der Verbindung Probleme auftreten, stellen Sie sicher, dass die entsprechenden Voraussetzungen erfüllt sind.

Umgebung konfigurieren

Wenn Sie lokale Anwendungen und Tools mit der Db2-Datenbank verbinden möchten, müssen Sie die Umgebung konfigurieren.

Programmgestützte Verbindungsherstellung

Für die Erstellung von Anwendungen, die eine Verbindung mit einer Db2-Datenbank herstellen, können die gängigen Programmiersprachen verwendet werden.

  • JDBC
  • ODBC
  • .NET
  • PHP

Verbindung für Apps und Tools herstellen

Sie können auch externe Anwendungen und Tools mit Db2 on Cloud verbinden und diese nutzen, um zusätzliche Verwaltungs- und Analyseaktionen für Ihre Daten durchzuführen. Beispiele:

  • Verbindung Ihrer IBM® Cloud-Anwendungen, die eine Analysedatenbank benötigen.
  • Verbindung von IBM Watson™ Studio (früher als IBM Data Science Experience bezeichnet).
  • Verbindung mit IBM InfoSphere® Data Architect für den Entwurf und die Bereitstellung des Datenbankschemas.
  • Verbindung eines IBM Cognos®-Servers zur Ausführung von Cognos-Berichten für Ihre Daten.
  • Verbindung mit SQL-basierten Tools wie Tableau oder Microsoft Excel für die Datenbearbeitung, -analyse oder -visualisierung.
    • Tableau
    • Microsoft Excel
  • Verbindung mit Aginity Workbench für die Migration von Netezza®-Datenmodellen und -Daten in Db2 Warehouse on Cloud.
