데이터베이스 호환성 평가

IBM Database Conversion Workbench(DCW)는 IBM Data Studio에 데이터베이스 마이그레이션 기능을 추가하는 무료 플러그인입니다. 웹 콘솔에서 Data Studio와 Database Conversion Workbench를 다운로드할 수 있습니다.

Db2를 사용하는 데이터베이스의 호환성을 평가하고 SQL을 변환하기 위해 IBM Database Harmony Profiler 라고 하는 무료 경량 도구를 사용할 수도 있습니다.

Database Conversion Workbench 및 Database Harmony Profiler에 대한 자세한 정보는 IBM Database Conversion Workbench(DCW) 를 참조하십시오.
