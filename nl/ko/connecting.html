연결

명령행 인터페이스, IBM® 또는 서드파티 애플리케이션이나 도구, 작성한 앱을 Db2® 데이터베이스에 연결할 수 있습니다.

전제조건

연결 중에 문제점이 발생하면 전제조건 을 완료하십시오.

환경 구성

Db2 데이터베이스에 로컬 애플리케이션 및 도구를 연결하려면 환경 구성 을 수행해야 합니다.

프로그래밍 방식으로 연결

공통 프로그래밍 언어를 사용하여 Db2 데이터베이스에 연결하는 애플리케이션을 작성할 수 있습니다.

  • JDBC
  • ODBC
  • .NET
  • PHP

앱 및 도구 연결

또한 외부 애플리케이션 및 도구를 Db2 on Cloud에 연결하고 이를 사용하여 데이터를 관리하고 분석할 수 있습니다. 예를 들어, 다음과 같습니다.

  • 분석 데이터베이스가 필요한 IBM® Cloud 애플리케이션을 연결합니다.
  • IBM Watson™ Studio(이전 이름:IBM Data Science Experience)에서 연결합니다.
  • IBM InfoSphere® Data Architect를 연결하여 데이터베이스 스키마를 디자인하고 배치합니다.
  • IBM Cognos® 서버를 연결하여 데이터에 대한 Cognos 보고서를 실행합니다.
  • Tableau 또는 Microsoft Excel과 같은 SQL 기반 도구를 연결하여 데이터를 조작하거나 분석하거나 시각화합니다.
    • Tableau
    • Microsoft Excel
  • Aginity Workbench를 연결하여 Netezza® 데이터 모델 및 데이터를 Db2 Warehouse on Cloud에 마이그레이션합니다.
