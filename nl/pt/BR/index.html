Introdução

O IBM® Db2® on Cloud é um banco de dados SQL provisionado para você na nuvem. É possível usar o Db2 on Cloud da mesma forma que usaria qualquer software de banco de dados, mas sem o gasto adicional e a despesa de configuração de hardware e instalação e manutenção de software.

Crie credenciais. Para aqueles novos no IBM Cloud, depois de criar o serviço, deve-se criar um nome de usuário e uma senha clicando no botão Criar credenciais ao ativar o serviço. Tecnicamente, é possível efetuar login no console da web sem credenciais, mas você precisará do nome do usuário e da senha para usar muitas das ferramentas do Db2.

Também é possível instalar um banco de dados local do Db2 usando o download gratuito do Db2 Developer Edition . Ele instala rapidamente uma edição do desenvolvedor pronta para execução do Db2 com ferramentas dentro de um contêiner do Docker (o Docker não é necessário; qualquer componente necessário será instalado automaticamente).

Interfaces

É possível trabalhar com o seu banco de dados do Db2 on Cloud das maneiras a seguir:

  • Console da Web do Db2 on Cloud
  • API do REST
  • Conecte aplicativos ou suas ferramentas favoritas por meio do seu computador local
  • Use o Db2 on Cloud como uma origem de dados para seus apps ou serviços IBM Cloud

Console da web do Db2 on Cloud

O console da web fornece uma interface gráfica para tudo o que precisar para usar seu banco de dados, incluindo: instalações de carga, um editor de SQL, downloads de driver e mais.

É possível acessar o seu console da web do Db2 on Cloud das maneiras a seguir:

  • No painel do IBM Cloud: é possível abrir o console da web por meio da página Detalhes do serviço para seu serviço do IBM Db2 on Cloud.
  • URL direta: é possível marcar como favorita a URL do console da web para seu serviço do Db2 on Cloud.

Instalando clientes da linha de comando e drivers do Db2 no computador

A maioria dos usuários pode ignorar essa etapa. Na maioria dos casos, os usuários tendem a usar apenas a API de REST ou instalar drivers para uma estrutura, por exemplo, com o comando pip da Python. No entanto, os usuários avançados podem desejar usar o cliente da linha de comando do Db2 para administrar seu banco de dados e usar os comandos do Db2. Além disso, determinados aplicativos ODBC ou JDBC podem se beneficiar de uma instalação geral de drivers do Db2. Se esse for o caso, conclua as etapas a seguir:

  1. Instale o pacote de drivers no computador em que seus apps ou ferramentas estão em execução.
  2. Configure os arquivos de drivers para seu banco de dados do Db2 on Cloud.

Use o Db2 on Cloud como uma origem de dados para os seus apps ou serviços do IBM Cloud

Os apps hospedados no IBM Cloud podem conectar-se ao seu banco de dados do Db2 on Cloud exatamente da mesma maneira que os seus aplicativos locais se conectam ao seu banco de dados do Db2 on Cloud.

Quando seus apps usam a plataforma IBM Cloud, é possível aproveitar a variável de ambiente VCAP _SERVICES para simplificar a tarefa de especificar detalhes e credenciais do banco de dados:

  1. Em seu painel do IBM Cloud, na guia Conexões da página Detalhes do serviço para seu serviço do IBM Db2 on Cloud, clique no botão Criar conexão.
  2. Selecione o app do IBM Cloud para usar com seu banco de dados do Db2 on Cloud como uma origem de dados e, em seguida, clique no botão Conectar.

  3. Atualize seu código do aplicativo para recuperar detalhes e credenciais do banco de dados por meio da variável de ambiente VCAP_SERVICES:

    Exemplo sem VCAP_SERVICES

     <?php
 $driver      = "DRIVER={IBM DB2 ODBC DRIVER};";

 $database    = "BLUDB";         # Get these database details from
 $hostname    = "<Host-name>";   # the Connect page of the Db2 on Cloud
 $port        = 50000;           # web console.
 $user        = "<User-ID >";    #
 $password    = "<Password>";    #
 $dsn         = "DATABASE=$database;" .
                "HOSTNAME=$hostname;" .
                "PORT=$port;" .
                "PROTOCOL=TCPIP;" .
                "UID=$user;" .
                "PWD=$password;";

 $conn_string = $driver . $dsn;

 $conn        = db2_connect( $conn_string, "", "" );
 ?>

    Exemplo com VCAP_SERVICES

     <?php
 $driver      = "DRIVER={IBM DB2 ODBC DRIVER};";

 $vcap        = json_decode( getenv( "VCAP_SERVICES" ), true );
 $dsn         = $vcap[ "dashDB" ][0][ "credentials" ][ "dsn" ];

 $conn_string = $driver . $dsn;

 $conn        = db2_connect( $conn_string, "", "" );
 ?>
