Tutorial de Introdução

O IBM® Db2® on Cloud é um banco de dados SQL que é provisionado para você na nuvem. É possível usar o Db2 on Cloud do mesmo modo que você usaria qualquer software de banco de dados, mas sem o tempo e a despesa de configuração de hardware ou instalação e manutenção de software.

Crie credenciais. Para aqueles que são novos no IBM Cloud, após criar o serviço, deve-se criar um nome do usuário e senha clicando no botão Criar credenciais ao iniciar o serviço. Tecnicamente, é possível efetuar login no console da web sem credenciais, mas você precisa do nome do usuário e da senha para usar muitas das ferramentas do Db2.

Também é possível instalar um banco de dados DB2 local usando o download grátis do DB2 Developer Edition. Ele instala rapidamente uma edição de desenvolvedor pronta para desenvolvimento do Db2 com ferramentas dentro de um contêiner do Docker (o Docker não é requerido; ele instala automaticamente quaisquer componentes necessários).

Interfaces

É possível trabalhar com o seu banco de dados do Db2 on Cloud das maneiras a seguir:

  • Console da Web do Db2 on Cloud
  • API do REST
  • Conecte aplicativos ou suas ferramentas favoritas por meio do seu computador local
  • Use o Db2 on Cloud como uma origem de dados para seus apps ou serviços IBM Cloud

Console da web do Db2 on Cloud

O console da web fornece uma interface gráfica para tudo o que precisar para usar seu banco de dados, incluindo: instalações de carga, um editor de SQL, downloads de driver e mais.

É possível acessar o seu console da web do Db2 on Cloud das maneiras a seguir:

  • No painel do IBM Cloud: é possível abrir o console da web por meio da página Detalhes do serviço para seu serviço do IBM Db2 on Cloud.
  • URL direta: é possível marcar como favorita a URL do console da web para seu serviço do Db2 on Cloud.

Instalando clientes de linha de comandos e drivers do Db2 em seu computador

Na maioria dos casos, os usuários tendem a usar apenas a API de REST ou instalar drivers para uma estrutura, por exemplo, com o comando pip da Python.

Para instalar pacotes de drivers Python para o MacOS, deve-se usar o comando pip com a opção --no-cache-dir. Para obter instruções detalhadas, veja: Suporte do Python para o IBM DB2 e IBM Informix

Se você estiver fazendo um trabalho de ciência de dados com o Python ou desejar suporte para quadros de dados, análise dentro do banco de dados ou usar o Watson Studio, pode desejar usar o driver alternativo a seguir, que se concentra no suporte a uma variedade de funções de ciência de dados: ibmdbpy 0.1.5.

Para instalar os drivers de estrutura Python ou Node.js, clique em um dos links a seguir:

  • Drivers Python
  • Drivers Node.js

Para instalar o Node.js ORM denominado Sequelize, clique no link a seguir: Sequelize

Para obter mais opções de instalação do driver, incluindo Java, Go, Jupyter Notebooks, Ruby, PHP e mais, clique no link a seguir:

  • ibmdb

Supondo que você já tenha instalado drivers de estrutura, é possível ignorar as etapas a seguir. No entanto, os usuários avançados podem desejar usar o cliente de linha de comandos do Db2 para administrar seu banco de dados e usar os comandos do Db2. Além disso, determinados aplicativos ODBC ou JDBC podem se beneficiar de uma instalação geral de drivers do Db2. Se esse for o caso, conclua as etapas a seguir:

  1. Instale o pacote de drivers no computador no qual seus apps ou ferramentas estão em execução.
  2. Configure os arquivos de driver para seu banco de dados do Db2 on Cloud.

Use o Db2 on Cloud como uma origem de dados para os seus apps ou serviços do IBM Cloud

Os apps que são hospedados no IBM Cloud podem se conectar ao seu banco de dados do Db2 on Cloud da mesma maneira que os aplicativos locais se conectam ao seu banco de dados do Db2 on Cloud.

Quando seus apps usam a plataforma IBM Cloud, é possível aproveitar a variável de ambiente VCAP _SERVICES para simplificar a tarefa de especificar detalhes e credenciais do banco de dados:

  1. Em seu painel do IBM Cloud, na guia Conexões da página Detalhes do serviço para seu serviço do IBM Db2 on Cloud, clique no botão Criar conexão.
  2. Selecione o app do IBM Cloud para usar com seu banco de dados do Db2 on Cloud como uma origem de dados e, em seguida, clique no botão Conectar.

  3. Atualize seu código do aplicativo para recuperar detalhes e credenciais do banco de dados por meio da variável de ambiente VCAP_SERVICES:

    Exemplo sem VCAP_SERVICES

     <?php
 $driver      = "DRIVER={IBM DB2 ODBC DRIVER};";

 $database    = "BLUDB";         # Get these database details from
 $hostname    = "<Host-name>";   # the Connect page of the Db2 on Cloud
 $port        = 50000;           # web console.
 $user        = "<User-ID >";    #
 $password    = "<Password>";    #
 $dsn         = "DATABASE=$database;" .
                "HOSTNAME=$hostname;" .
                "PORT=$port;" .
                "PROTOCOL=TCPIP;" .
                "UID=$user;" .
                "PWD=$password;";

 $conn_string = $driver . $dsn;

 $conn        = db2_connect( $conn_string, "", "" );
 ?>

    Exemplo com VCAP_SERVICES

     <?php
 $driver      = "DRIVER={IBM DB2 ODBC DRIVER};";

 $vcap        = json_decode( getenv( "VCAP_SERVICES" ), true );
 $dsn         = $vcap[ "dashDB" ][0][ "credentials" ][ "dsn" ];

 $conn_string = $driver . $dsn;

 $conn        = db2_connect( $conn_string, "", "" );
 ?>
