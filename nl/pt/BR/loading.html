Carregando dados

É possível carregar dados de um arquivo de dados em um formato delimitado, como CSV ou TXT localizado em uma rede local, um armazenamento de objeto (Amazon S3 ou IBM Cloud Object Storage (anteriormente Swift SoftLayer)) ou usando as ferramentas do DB2®. Também é possível preencher uma instância de banco de dados com dados executando um processo de carregamento de um aplicativo como o InfoSphere® DataStage®.

  • Carregando dados com a CLI Lift
  • Carregando dados do Oracle
  • Carregando dados do SQL Server
  • Carregando um arquivo CSV

  • Carregue dados do Amazon S3 usando um dos métodos a seguir:

    • Console da web do Db2 Warehouse on Cloud. Carregar > Amazon S3.

    • Tabelas externas diretamente. A seguinte é uma instrução SQL de exemplo:

      INSERT INTO <table-name> SELECT * FROM EXTERNAL '<mys3file.txt>' USING
  (CCSID 1208 s3('s3.amazonaws.com', 
  '<S3-access-key-ID>',
  '<S3-secret-access-key>', 
  '<my_bucket>'
     )
  )
