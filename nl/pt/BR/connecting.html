Conectando

É possível conectar interfaces da linha de comandos, aplicativos ou ferramentas da IBM® ou de terceiros ou aplicativos que você cria para o banco de dados DB2®.

Prerequisites

Se você estiver com problemas para se conectar, conclua ospré-requisitos.

Configurando seu ambiente

Para conectar aplicativos locais e ferramentas ao banco de dados DB2, é necessário configurar seu ambiente.

Conectando programaticamente

É possível usar linguagens de programação comuns para criar aplicativos que se conectam a um banco de dados DB2.

  • JDBC
  • ODBC
  • .NET
  • PHP

Conectando aplicativos e ferramentas

Também é possível conectar ferramentas e aplicativos externos ao Db2 on Cloud e utilizá-los para gerenciar ou analisar ainda mais os dados. Por exemplo:

  • Conecte seus aplicativos IBM Cloud™ que precisam de um banco de dados de análise.
  • Conecte-se do IBM Watson™ Studio (anteriormente IBM Data Science Experience).
  • Conecte-se ao IBM InfoSphere® Data Architect para projetar e implementar seu esquema de banco de dados.
  • Conecte-se a um servidor IBM Cognos® para executar relatórios Cognos com relação aos seus dados.
  • Conecte ferramentas baseadas em SQL, como Tableau ou Microsoft Excel, para manipular, analisar ou visualizar seus dados.
    • Tableau
    • Microsoft Excel
  • Conecte-se com o Aginity Workbench para migrar modelos de dados e dados do Netezza® para o Db2 Warehouse on Cloud.
