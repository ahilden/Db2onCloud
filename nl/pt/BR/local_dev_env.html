Ambiente de desenvolvimento local

Se você deseja configurar um ambiente de desenvolvimento DB2 local, é possível usar qualquer um dos seguintes:

  • IBM DB2 Developer Community Edition Grátis. Inclui todos os recursos do DB2, mas apenas para uso de desenvolvimento.
  • IBM DB2 Express-C Grátis. Uma versão limitada do DB2 que é grátis para qualquer uso, incluindo sistemas de produção ou integrados.
