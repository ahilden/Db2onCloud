Avaliando a compatibilidade do seu banco de dados

O IBM Database Conversion Workbench (DCW) é um plug-in gratuito que inclui recursos de migração do banco de dados no IBM Data Studio. É possível fazer download do Data Studio e do Database Conversion Workbench por meio do console da web.

Para avaliar a compatibilidade do seu banco de dados com o Db2 e converter o SQL, também é possível usar a ferramenta leve e grátis denominada IBM Database Harmony Profiler .

Para obter mais informações sobre o Database Conversion Workbench e o Database Harmony Profiler, veja: IBM Database Conversion Workbench (DCW) .
