免费套餐

IBM® Db2® on Cloud 免费套餐提供了基本资源，可用于开发或了解 Db2，无需任何费用。

免费套餐无时间限制，但用户必须每 30 天重新延长其免费套餐。

仅提供社区支持。

体系结构

与其他 Db2 on Cloud 套餐不同，Db2 on Cloud 免费套餐在多租户系统上运行。Flex 和 Precise Performance 套餐在其自己的单租户虚拟机或裸机服务器上运行。

免费套餐允许使用一个数据库模式。

有关 Db2 on Cloud 免费套餐的更多信息，请参阅常见问题。

在北美大陆之外使用免费套餐

对于北美大陆，目前无需信用卡也可以访问免费套餐。在北美大陆之外，客户必须向其 IBM Cloud 帐户添加信用卡，然后选择“美国南部 - 达拉斯”区域，才能查看和选择免费套餐。

免费套餐始终免费且不会向信用卡收取费用。
