评估数据库兼容性

IBM Database Conversion Workbench (DCW) 是一个免费插件，它向 IBM Data Studio 增加了数据库迁移功能。您可以从 Web 控制台下载 Data Studio 和 Database Conversion Workbench。

要评估数据库与 Db2 的兼容性并转换 SQL，还可以使用免费的轻量级工具 IBM Database Harmony Profiler 。

有关 Database Conversion Workbench 和 Database Harmony Profiler 的更多信息，请参阅：IBM Database Conversion Workbench (DCW) 。
