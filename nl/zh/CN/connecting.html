连接

您可以将命令行界面、IBM® 或第三方应用程序或工具，或者自己创建的应用程序连接到 Db2® 数据库。

先决条件

如果无法连接，请完成先决条件 。

配置环境

要将本地应用程序和工具连接到 Db2 数据库，您需要配置环境 。

以编程方式连接

您可以使用常用编程语言来创建连接到 Db2 数据库的应用程序。

  • JDBC
  • ODBC
  • .NET
  • PHP

连接应用程序和工具

您还可以将外部应用程序和工具连接到 Db2 on Cloud，以使用其进一步管理或分析数据。例如：

  • 连接需要分析数据库的 IBM® Cloud 应用程序。
  • 从 IBM Watson™ Studio（以前称为 IBM Data Science Experience）连接。
  • 连接 IBM InfoSphere® Data Architect，以设计和部署数据库模式。
  • 连接 IBM Cognos® 服务器，以对数据运行 Cognos 报告。
  • 连接基于 SQL 的工具（如 Tableau 或 Microsoft Excel），以控制、分析或可视化数据。
    • Tableau
    • Microsoft Excel
  • 连接 Aginity Workbench，以将 Netezza® 数据模型和数据迁移到 Db2 Warehouse on Cloud。
