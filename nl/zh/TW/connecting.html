連接

您可以將指令行、IBM® 或協力廠商應用程式或工具，或是您建立的應用程式，連接到您的 Db2® 資料庫。

必要條件

如果您在連接時有問題，請完成必要條件 。

配置環境

若要將本端應用程式及工具連接至 Db2 資料庫，您需要配置環境 。

以程式設計方式連接

您可以使用一般程式設計語言來建立連接 Db2 資料庫的應用程式。

  • JDBC
  • ODBC
  • .NET
  • PHP

連接應用程式及工具

您也可以將外部應用程式及工具連接至 Db2 on Cloud，並使用它們來進一步管理或分析資料。例如：

  • 連接需要分析資料庫的 IBM® Cloud 應用程式。
  • 從 IBM Watson™ Studio（先前稱為 IBM Data Science Experience）連接。
  • 連接 IBM InfoSphere® Data Architect 以設計及部署資料庫綱目。
  • 連接 IBM Cognos® 伺服器以對資料執行 Cognos 報告。
  • 連接 SQL 工具，例如 Tableau 或 Microsoft Excel，以操作、分析或視覺化您的資料。
    • Tableau
    • Microsoft Excel
  • 連接 Aginity Workbench 以將 Netezza® 資料模型和資料移轉至 Db2 Warehouse on Cloud。
