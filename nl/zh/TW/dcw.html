評量資料庫相容性

IBM Database Conversion Workbench (DCW) 是免費的外掛程式，為 IBM Data Studio 新增了資料庫移轉功能。您可以從 Web 主控台下載 Data Studio 和 Database Conversion Workbench。

若要評量資料庫與 Db2 的相容性，並轉換 SQL，您也可以使用免費的小型工具，稱為 IBM Database Harmony Profiler 。

如需 Database Conversion Workbench 和 Database Harmony Profiler 的相關資訊，請參閱：IBM Database Conversion Workbench (DCW) 。
