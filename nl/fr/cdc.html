Capture de données de changement (CDC) InfoSphere

Plusieurs méthodes permettent de connecter une base de données Db2 à IBM® Db2® on Cloud. L'une consiste à capturer les données qui contiennent des modifications dans votre base de données Db2 sur site par capture de données de changement (CDC) IBM InfoSphere, qui est une solution de synchronisation des données temps réel.

Pour synchroniser vos données sur site en temps réel sur votre base de données Db2 on Cloud à l'aide de CDC InfoSphere, voir ce tutoriel : How to stay online while moving data to Db2 on Cloud.
