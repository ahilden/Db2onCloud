Connexion

Vous pouvez connecter à votre base de données Db2® des interfaces de ligne de commande, des applications ou des outils tiers ou IBM®, ou des applications que vous créez.

Prérequis

Si vous rencontrez des problèmes de connexion, consultez l'article Prérequis .

Configuration de votre environnement

Pour connecter des applications et des outils locaux à votre base de données Db2, vous devez configurer votre environnement .

Connexion à l'aide d'un programme

Vous pouvez utiliser des langages de programmation courants pour créer des applications qui se connectent à une base de données Db2.

  • JDBC
  • ODBC
  • .NET
  • PHP

Connexion d'applications et d'outils

Vous pouvez aussi connecter des outils et des applications externes à Db2 on Cloud et les utiliser pour gérer ou analyser vos données. Exemple :

  • Connectez vos applications IBM Cloud™ qui nécessitent une base de données analytique.
  • Etablissez une connexion à partir d'IBM Watson™ Studio (anciennement IBM Data Science Experience).
  • Connectez IBM InfoSphere® Data Architect pour concevoir et déployer votre schéma de base de données.
  • Connectez un serveur IBM Cognos® pour exécuter des rapports Cognos avec vos données.
  • Connectez des outils basés sur SQL comme Tableau ou Microsoft Excel pour manipuler, analyser ou visualiser vos données.
    • Tableau
    • Microsoft Excel
  • Connectez Aginity Workbench pour migrer des données et des modèles de données Netezza® vers Db2 Warehouse on Cloud.
