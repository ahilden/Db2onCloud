Conexión

Puede conectar interfaces de línea de mandatos, aplicaciones o herramientas de IBM® o de otros proveedores o apps que cree con su base de datos Db2®.

Requisitos previos

Si tiene problemas a la hora de conectarse, asegúrese de cumplir con los requisitos previos .

Configuración del entorno

Para conectar aplicaciones y herramientas locales a la base de datos Db2, debe configurar el entorno .

Conexión mediante programas

Puede utilizar los lenguajes de programación más utilizados para crear aplicaciones que se conecten a una base de datos Db2.

  • JDBC
  • ODBC
  • .NET
  • PHP

Conexión de apps y herramientas

También puede conectar aplicaciones externas a Db2 on Cloud y utilizarlas para gestionar o analizar todavía más sus datos. Por ejemplo:

  • Conecte sus aplicaciones IBM® Cloud que necesitan una base de datos de análisis.
  • Conecte desde IBM Watson™ Studio (antes denominado IBM Data Science Experience).
  • Conecte IBM InfoSphere® Data Architect para diseñar y desplegar su esquema de base de datos.
  • Conecte un servidor IBM Cognos® para ejecutar informes de Cognos de sus datos.
  • Conecte herramientas basadas en SQL, como Tableau o Microsoft Excel, para manipular, analizar o visualizar sus datos.
    • Tableau
    • Microsoft Excel
  • Conecte Aginity Workbench para migrar modelos de datos y datos de Netezza® a Db2 Warehouse on Cloud.
