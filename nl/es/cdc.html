InfoSphere CDC

Hay muchas formas de conectar una base de datos DB2 a IBM® Db2® on Cloud. Una forma es capturar datos que han cambiado en la base de datos DB2 en sus instalaciones con IBM InfoSphere Change Data Capture (InfoSphere CDC), que es una solución de sincronización de datos en tiempo real.

Para sincronizar sus datos en las instalaciones locales en tiempo real con su base de datos de Db2 on Cloud mediante InfoSphere CDC, consulte esta guía de aprendizaje: Cómo mantenerse en línea mientras se mueven datos a Db2 en Cloud.
