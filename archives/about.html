About Db2 on Cloud

IBM® Db2® on Cloud is a fully managed SQL database service. The Db2 on Cloud web console makes common tasks easy, such as importing spreadsheets, running SQL statements, and monitoring your service.

What is included in the managed service?

The fully managed service of Db2 on Cloud deploys in a few clicks and you can run your database without hassles. It's already optimized for you. The service handles any underlying software upgrades and hardware issues for you. The service includes 24x7 health monitoring of the database and infrastructure. Your on-disk data is encrypted and connections to the database are made with enhanced security. High availability plans provide seamless rolling updates and you can you scale your apps without downtime. For more information, see High Availability.
