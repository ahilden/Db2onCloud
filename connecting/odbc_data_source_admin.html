Connecting with the ODBC Data Source Administrator

Use the Microsoft ODBC Data Source Administrator tool to define a connection between an ODBC or CLI application and a Db2 on Cloud database.

Prerequisites

Before attempting to connect to your Db2 on Cloud database, verify that you have the necessary prerequisites.

Procedure

  1. Install the Db2 driver package.

  2. Open ODBC Data Source Administrator and create either a User DSN or System DSN for the Db2 driver package.

  3. Click Advanced Settings. Enter following CLI parameters with their values for the Db2 on Cloud server: Hostname, Port, and Database.

  4. For an SSL connection, enter CLI parameter Security with value SSL.

  5. Click Apply.

  6. To test the connection, return to main page of ODBC Data Source Administrator. Click Configure for the DSN that you created. Enter the user ID and password for the server and click Connect.
